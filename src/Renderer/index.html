Stream with OBS Studio

Wingman Player streams video through the player window and audio through a dedicated localhost feed only OBS can reach. Add two sources in OBS:

1. Video: Window Capture

  1. Add a Window Capture source.
  2. Set Window to Wingman Player.
  3. Set Capture Method to Windows 10 (1903 and up).
  4. Untick Capture Cursor.
  5. Tick Client Area.
  6. Leave Capture Audio (BETA) unticked. Audio comes from the next source.

2. Audio: Media Source

  1. Add a Media Source.
  2. Untick Local File.
  3. Paste this into Input: http://127.0.0.1:17329/stream.wav
  4. Tick Restart playback when source becomes active.

Press F8 to hide the player from your monitor. Viewers still see it in your stream.
