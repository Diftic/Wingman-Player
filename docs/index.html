Wingman Player.
A YouTube overlay for Windows.

Wingman Player is a sci-fi framed YouTube overlay that lives at the edge of your screen. Driven by the Wingman YouTube skill, always on top, out of the way until you need it.

Installer (.msi) Portable (.exe)

What it does

A focused media overlay. Press your hotkey, drop a video in via the Wingman skill, fly.

Sci-fi frame

Transparent, always-on-top window with a 16:9 video cutout. Drag anywhere, resize from 30% to 100%.

Instant toggle

Default hotkey F8 shows and hides the overlay without touching the mouse. Rebind to any combo.

Mini banner

Compact "now playing" tile in the lower-right while the overlay is hidden. Always know what's playing.

OBS streaming

Frame is capturable via Window Capture. Audio is exposed at http://127.0.0.1:17329/stream.wav for an OBS Media Source. No third-party audio routing.

Position memory

The window remembers exactly where you left it between sessions.

Auto-update

Checks GitHub Releases on launch and offers an in-place update. Manual recheck button in Settings.

Driving the player

Load any YouTube video or playlist into the overlay from the host:

// Load a single video
window.__wingmanLoad('dQw4w9WgXcQ');

// Load a playlist
window.__wingmanLoad(null, 'PLxxxxxxxxxxxxxxxxxx');

// Return to the idle state
window.__wingmanLoad(null, null);
